## Features

- User-friendly game interface with an easy-to-navigate menu.
- Seamless transitions between game states: menu, game screen, and tutorial.
- The design is responsive and optimized for desktop use.

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage
